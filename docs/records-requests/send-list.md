# RoofSonar — Records Request Send List (all 38 jurisdictions)

Researched 2026-06-10 against official .gov/.org/.fl.us sources only. City of Orlando is excluded — its permit data already flows in via the Socrata API.

**How to use this list**
1. Send the email template below to the **Send to** address; add the **CC** when listed.
2. Where the row says **PORTAL**, the jurisdiction only accepts requests through their online portal — paste the same template text into the portal form instead of emailing.
3. Rows marked ⚠️ need a quick phone confirmation before sending (details in Notes).
4. Florida Ch. 119 does not require you to identify yourself or your purpose — the template stays minimal on purpose. Fees must be quoted before fulfillment; the template asks for that.

---

## Email template

> **Subject:** Public Records Request — Roofing Permit Data
>
> To the Records Custodian:
>
> Per Florida Statutes Chapter 119, I request the following in electronic format (CSV or Excel preferred):
>
> 1. All issued roofing/re-roof permits from January 1, 2000 to present, including: permit number, parcel ID, site address, permit type/work description, application date, issue date, final date, and status.
> 2. The same report monthly going forward (the prior month's issued roofing permits) as a standing request, if your office can accommodate it.
>
> Please advise of any fees before fulfilling. If any portion of this request is unclear or would be easier to fulfill in a different format, I'm happy to adjust — a raw export from your permitting system works fine.
>
> Thank you.
>
> [YOUR NAME]
> [COMPANY]
> [PHONE] · [EMAIL]

For portal submissions, use the numbered request body (items 1–2 plus the fee line) as the request description.

## Sending strategy

1. **Wave 1 (highest housing stock, High confidence):** Seminole County, Volusia County, Orange County, Sanford, Oviedo, Altamonte Springs, Winter Springs, Casselberry, Longwood, Lake Mary, Deltona (portal), Daytona Beach, Port Orange (portal), Ormond Beach, DeLand, New Smyrna Beach, Winter Park, Apopka, Ocoee, Winter Garden, Maitland-adjacent cities.
2. **Wave 2 (small towns):** Edgewater, DeBary, Orange City, Holly Hill, Daytona Beach Shores, Ponce Inlet, Lake Helen, Pierson, Belle Isle, Edgewood, Eatonville, Oakland.
3. **Phone-first (3):** Windermere (no published clerk email), Oak Hill (verify clerk email), Maitland (confirm routing). South Daytona can be sent via portal without a call.
4. **Skip/last (2):** Bay Lake, Lake Buena Vista — Disney property, ~zero residential roofs.

When extracts arrive: drop the CSV into `data/inbox/<slug>/`, create `ingest/configs/<slug>.json` from `_example.json`, and run `npm run ingest:file -- <slug> <file>`. The 39 per-jurisdiction drafts in `docs/records-requests/drafts/` remain available if you prefer per-city letters; this list supersedes their "TBD" recipient lines.

**Custodian-vs-contractor note:** Five jurisdictions contract building services out (Oak Hill → UES, Belle Isle → UES, DeBary → PDCS, Windermere → PDCS, Oakland → Willdan; Edgewood → Orange County). Under Ch. 119 the municipality remains the records custodian — always send to the city/town clerk, CC the contractor.
